# Data Cleaning & Structural Validation

## Objective
Transform raw data into a reliable format for analysis.

## Data Issues Found
- Duplicate records
- Missing values
- Inconsistent date formats

## Cleaning Steps
1. Removed duplicate entries.
2. Filled missing names with "Unknown".
3. Filled missing salary values.
4. Standardized all dates to YYYY-MM-DD format.
5. Validated data consistency.

## Files
- raw_data.csv
- cleaned_data.csv

## Outcome
The dataset is now clean, consistent, and ready for analysis.
